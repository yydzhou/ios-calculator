# iOS Calculator

A SwiftUI calculator app modeled after the Apple iOS Calculator, targeting the App Store.

## Requirements

- Xcode 15+
- iOS 17+
- Swift 5.9+

## Structure

- `Calculator/Sources/` — App source (SwiftUI views + ViewModel)
- `CalculatorTests/` — Unit tests for `CalculatorViewModel`

## Features

- Basic arithmetic: addition, subtraction, multiplication, division
- Percentage and sign toggle
- Matches the look and feel of the native iOS Calculator
